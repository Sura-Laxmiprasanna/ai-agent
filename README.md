🚀 AI Agent using Playwright & LLM
📌 Overview

This project is an AI-powered browser automation agent that performs tasks like searching, browsing websites, and extracting information automatically. It integrates Large Language Models (LLMs) with browser automation to execute intelligent actions through a user-friendly web interface.

🤖 AI-based task execution
🌐 Automated web browsing using Playwright

🧠 Integration with LLM APIs (OpenAI / Gemini)

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/Sura-Laxmiprasanna/ai-agent.git
cd ai-agent
2️⃣ Create Virtual Environment (Optional)
python -m venv venv
venv\Scripts\activate
3️⃣ Install Dependencies
pip install -r requirements.txt
4️⃣ Install Playwright Browsers
playwright install
5️⃣ Configure API Keys

Create a .env file in the root directory and add:

OPENAI_API_KEY=your_api_key_here
# OR
GOOGLE_API_KEY=your_api_key_here
6️⃣ Run the Application
python webui.py --ip 127.0.0.1 --port 7788
